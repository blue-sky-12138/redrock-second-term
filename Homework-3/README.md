# 注意事项

1. clientPart里放的是客户端，serverPart放的是服务端，服务端已部署到服务器上，若想更改客户端连接URL，请改客户端的server/client.go里的const地址常量
2. 由于命令行不能选择文件(我不会)，若要传输文件，请手动更改客户端的server/client.go第133行附近，手动取消注释与注释
3. 屏蔽与敏感词的更改请自行连接至数据库进行更改
